到了晚上是郭宇老师的分享会，主题是《零知识证明》，也就是zkp（zero knowledge proof）；郭宇老师说zkp里面最关键的是这个p，而不是zk。

下面抄一下Demian德米安的作业：放在另外的笔记上了
我的总结：
关键是p 不等于 np这个公式

p就好比尼古拉斯特斯拉
np就类比普罗大众

如果当p被np包含，或者说等于的话；那么意味着人类就没有存在的意义；因为当人的创造性可以通过复数堆叠来替代的话，那么人类的创造性就可以直接被计算机取代了。所以目前来说当前的话没有例子可以证明p=np，大家都认为p不等于np的。

一开始玩了小波哥主持了一场游戏来促进参会者的互相的认识
ps:其实第一天来的人根本我都认不太全
游戏规则如下：
大家站起来围成一个圈；从a->b->c->a;
然后依次发言:
a:我是a；
b:我是a旁边的b；
c:我是a旁边的b旁边的c。
到现场的人其实有20个人左右，经过一轮游戏之后，我大概把每个人的称呼和相貌对应了下来。
挺有意思的游戏。
后面讲了黄金圈法则：

要把一个东西快速的向别人描述出来，可以遵循这个法则：

  1. 先说你发现了的那个痛点是什么，用户是什么，这是why；
  2. 然后你在阐述你的方案是什么，或者说你的解决方法是什么；这是how；
  3. 最后才开始介绍你的产品；也就是说你最后坐了个什么东西，这是what。

Day2

11月30日，除了呆着大理本地的hacker以外，我和小波哥应该是第一批从外地来到大理参加hackerhouse的hacker。
早上的大理天空很漂亮，但是气温稍低，不时会有风吹来。我和小波哥早餐去吃了一份饵丝，味道还不错。
接着就逛了下大理的古城，不知道是因为疫情还是我们起的太早了。大多数的店面都没有开门。

里面有很多特色的小店，以及古色古香的街道。仿佛回到了小时候走在赤水的石塔塔的老街上。

中午，小波哥带着我一起去拜访了一位东北老大哥。因为他是住在靠近苍山的那边，而我们住在古城南门附近且地势较低。

我们一路向上，苍山逐渐占据了我所有的视野，不知为何，骤然让我产生了一种高山仰止的激动。大约十几分钟后，我们来到了老大哥所在的小区，两旁有着半开着的樱花树，以及不时会有松鼠穿过马路,仿佛回到了从高楼林立城市回到了大自然里。


老大哥一家都非常热情，让我一个有点社恐的人都被感染了。到他家里后，他和小波哥不一会就对创新教育学校展开了讨论。头一次听说混龄学校，教材都是自己编什么的，还有与传统学校不同的教育理念。后面又来了两位来交流的朋友：小A和李。她们又开始交流与孩子一起发生的趣事。


下午3点，和小波哥离开了老大哥的家里，去拜访面包树space拜访；初次见到了Jay杰，参观了他的食物森林DAO；老实说，我对Dao组织是没有什么概念的。因为Jay杰对Dao的结构已经有了一个十分深刻的理解，上来就跟我们讲他对于DAO构思之类，对我们进行了疯狂的输出，让我这个不太了解DAO的人一脑子浆糊，让我有了一点基础的了解；也让我迫切的想要汲取相关方面的知识。

接着后面又介绍了他的投资方案，让我一个不懂投资的人觉得十分有意思；简单来说就是4个池子：风险池，稳定池，周期池，现金流池。他命名为生存系统，动态化的按比例分配你的资金，保证自己不会一下子归0。

RSS是啥玩意

发表于 2022-07-04 更新于 2023-01-27 分类于 思考

起因

某天，我在逛别人的博客学习的时候，发现了一个的按钮。

然后出于好奇心我就点击了这个按钮，然后跳转到了一个全是代码的页面。

This XML file does not appear to have any style information associated with it. The document tree is shown below.

<rss xmlns:atom="http://www.w3.org/2005/Atom" xmlns:dc="http://purl.org/dc/elements/1.1/" version="2.0">

于是我没搞懂这些东西用来做啥的就去查了一下相关的东西

结果

在维基百科上查到的：

RSS（Really Simple Syndication），中文译作简易信息聚合，也称聚合内容，是一种消息来源格式规范，用以聚合多个网站更新的内容并自动通知网站订阅者。使用 RSS 后，网站订阅者便无需再手动查看网站是否有新的内容，同时 RSS 可将多个网站更新的内容进行整合，以摘要的形式呈现，有助于订阅者快速获取重要信息，并选择性地点阅查看。

以我不聪明的大脑来理解的话，RSS就是一个网站信息摘要聚合。是一种XML的格式规范，但是需要使用相应的阅读器才能够正常显示。不然就会是上面的一堆代码。

运动带给我什么

发表于 2022-06-28 更新于 2023-01-27 分类于 生活


好久没写博客了0^0!

从5月初到6月末，历时将近2个月；我从一开始的断断续续的跑2KM，然后慢慢增加路程，减少休息的时间；到现在能够5KM跑33minutes左右(虽然还是很菜；)

有了一定进步！！

运动让我体重从75kg多，到目前的72kg多，没有控制饮食；运动让我吃得更香，更好！身体更加的紧实！

同时也让我的精神更加的坚韧，跑到2KM，想着再跑远一点，然后就跑到了3KM；然后不断增加我的增援路程！晚上也睡得更香，调整了我晚上的生物钟(晚上老是失眠)。

在跑步过程中，看着河对岸的霓虹闪烁，路旁的花朵；让我的心情都变得愉悦了起来。

不过运动之前要记得热身，不然的话就容易伤到自身(过来人的经历)

总之一句话，运动让我变得更加热爱运动，热爱生活。^-^
